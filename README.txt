1. Clone the public git repo at https://github.com/karanssj4/ml-sp20--ass1 and move inside the directory
2. Make sure to have the latest version of python3 and pip installed
3. create a vitual environement or install the following packages using pip globally
	pip install sklearn numpy pandas seaborn
4. running `python script.py` should generate all the graphs that you see in the assignment

NOTE: In order to save execution time, non-best hyperparameters for GridSearchCV have been commented out. In order to check the GridSearchCV result, set do_whole_grid_search = True

5. Models will run iteratively, and you'd de able to see the confusion matrix, train and test times in console output.
